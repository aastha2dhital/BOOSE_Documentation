Class AppCommandFactory
Provides command creation logic for all custom commands used by the BOOSE application. This factory intercepts command keywords and returns the appropriate command objects.
Syntax
C#
    public class AppCommandFactory : CommandFactory

Documentation Preview
Constructor AppCommandFactory
Initialises a new instance of AppCommandFactory. No manual command registration is performed because this BOOSE version resolves commands through MakeCommand(string).
Syntax
C#
    public AppCommandFactory() : base()

Documentation Preview
Method MakeCommand
Creates command instances based on the command keyword provided in the source program. Custom commands such as CLEAR and RESET are defined here, while unhandled commands fallback to the base BOOSE command factory.
Syntax
C#
    public override ICommand MakeCommand(string commandType)
Parameters
commandType
The lowercase command keyword supplied in the program.
Return Value
A concrete ICommand matching the command keyword, or a base BOOSE command if no custom match is found.
